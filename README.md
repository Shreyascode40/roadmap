# ExamRoadmap

Complete preparation roadmaps for IIT JEE and NEET exams with structured timelines, study resources, and progress tracking to help students achieve their engineering and medical dreams.

## Features

- **JEE Preparation Roadmap** - Complete study plan for JEE Main & Advanced with 4-phase timeline
- **NEET Preparation Roadmap** - Comprehensive guide with Biology deep-dive and subject breakdown
- **Subject-wise Syllabus** - Physics, Chemistry, Mathematics (JEE) and Biology (NEET)
- **Study Resources** - Curated book recommendations and online platforms
- **Exam Calendar** - Important dates and countdown timers
- **Success Stories** - Motivation and strategies from successful candidates

## Quick Start

Open `index.html` in any modern web browser to view the roadmap.

```bash
# Clone the repository
git clone https://github.com/yourusername/roadmap.git

# Open in browser
start index.html
```
